# Global-comparison-of-women-in-parliament-throughout-the-20th-century-
# Overview

This project analyses the comparison of women in parliament throughout the 20th century across different continents. The raw dataset was collected from the Our World in Data website (World Health Organization, 2025). It can produce a HTML report.

# Research Question

Global comparison of women in parliament throughout the 20th century
